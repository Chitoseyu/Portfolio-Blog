## Getting Started

專案下載後請確認有安裝 Node.js

本地端可以透過執行以下指令運行

安裝依賴套件(node 預設只有 npm , pnpm需另外安裝)
```bash
npm install -g pnpm
```
透過pnpm執行安裝
```bash
pnpm install 
```
安裝完啟動本地端
```bash
pnpm dev
```
